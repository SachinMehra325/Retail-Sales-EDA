# 🛒 Retail Sales EDA Project

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on retail sales data to uncover patterns, trends, and business insights.

---

## 🎯 Objectives
- Understand dataset structure
- Clean and preprocess data
- Analyze sales performance
- Identify trends and patterns
- Generate business insights

---

## 🧠 What is EDA?
Exploratory Data Analysis (EDA) is the process of analyzing datasets using statistical and visualization techniques to understand patterns and extract insights.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Kaggle
- AI Tools
- Reasearch

---

## 🧹 Data Cleaning Steps
- Removed duplicate records
- Handled missing values
- Standardized text data
- Fixed invalid numeric values
- Renamed columns for clarity

---

## ⚙️ Feature Engineering
- Created **Profit_Tier** (Low, Medium, High)
- Extracted **Month & Year** from date

---

## 📊 Key Analysis
- Category-wise sales performance
- Top-selling products
- Payment method distribution
- Location-based sales comparison
- Monthly sales trends

---

## 📈 Visualizations
- Correlation Heatmap
- Monthly Sales Trend
- Category-wise Bar Chart
- Scatter Plot Analysis

---

## 💡 Key Insights
- Quantity strongly impacts total sales
- High-performing categories drive revenue
- Customer payment behavior varies
- Sales show monthly variation
- Some products underperform consistently

---

## 📌 Conclusion
EDA helps transform raw data into meaningful insights that support better business decisions.
